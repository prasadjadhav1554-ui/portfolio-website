# Assets Directory

This directory contains all the assets for Prasad Jadhav's Portfolio Website.

## Directory Structure

```
assets/
├── images/           # Profile photos, project screenshots, etc.
├── resume.pdf        # Downloadable resume file
└── README.md         # This file
```

## Instructions

### Adding Your Resume
1. Save your resume as a PDF file named `resume.pdf` in this directory
2. The "Download Resume" button in the hero section will automatically link to this file
3. Recommended: Keep the file size under 500KB for faster loading

### Adding Images
1. Place your profile photo in the `images/` directory
2. Name it appropriately (e.g., `profile.jpg`, `profile.png`)
3. Update the image source in `index.html` if you want to use a local image instead of the DiceBear avatar

### Recommended Image Sizes
- Profile Photo: 400x400px (square) or 300x400px (portrait)
- Project Screenshots: 800x500px (for best display quality)

## File Naming Conventions
- Use lowercase letters and hyphens for file names
- Examples: `profile-photo.jpg`, `project-screenshot.png`, `certificate-1.pdf`
